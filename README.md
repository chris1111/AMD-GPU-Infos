# AMD-GPU-Infos

### Simple utility menus to display Info of AMD Graphics Cards
- Working for Big Sur 11 and Monterey 12

#### Kexts required:
- Lilu.kext
- VirtualSMC.kext- SMCProcessor.kext- SMCSuperIO.kext
- ACPIMonitor.kext

### Supported cards:

#### 400 Series:
- RX 480
- RX 470D
- RX 470
- RX 460

#### 500 Series:
- RX 590
- RX 580X
- RX 580
- RX 570X
- RX 570
- RX 560X
- RX 560
